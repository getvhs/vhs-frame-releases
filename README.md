# vhs-frame-releases

download binaries and the auto-update feed for
[vhs-frame](https://github.com/getvhs/vhs-frame), a local AI generation canvas
for macOS.

## download

grab the latest `.dmg` from the
[releases page](https://github.com/getvhs/vhs-frame-releases/releases/latest).
apple silicon only.

## auto-update

installed copies check this repo for new releases and update themselves in the
background. no action needed.

## publishing (maintainers)

releases are cut from the vhs-frame source repo with `npm run release`.
